1) Download the protein sequences to <organism_name>.txt. If needed use add_file_extension.sh shell script (takes only exension, adds axtensions to all the files present in the current directory),
2) Run perl run_signal.pl organism_name (takes organism_name.txt files. Produces organism_name_all.out file with signal peptides predictions.
3) Run perl sigp4_analyzer.pl organism_name (takes organism_name_all.out, ensembl_parsed_organism_name. Produces sigp_ALL_organism_name.out and sigpL_ALL_organism_name.out),
4) Run perl Needle_wrapper.pl merged_organism_name.tsv sigpL_ALL.out organism_of_interest_name other_organisms_names (takes ortho mapper in tsv file, sigpL_ALL.out [beforehand it needs to be merged form outputs produced in 3rd step], ensembl_parsed_organism_name),
5) Run prep2comp.pl ALL_organism_name_Ortho DOKONCZ!
